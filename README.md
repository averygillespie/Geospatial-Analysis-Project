Geospatial Analysis of Event Density and Infrastructure Proximity
Project Overview

This project analyzes the geographic distribution of reported events and their proximity to major infrastructure across the United States, Canada, and Australia. The objective was to investigate spatial patterns, measure nearest-neighbor distances, and visualize density relationships using geospatial techniques.

Objectives

Clean and filter large-scale geographic datasets

Integrate multi-country infrastructure data

Compute nearest infrastructure distances using the Haversine formula

Analyze spatial density patterns across regions

Visualize geospatial relationships through comparative mapping

Data Sources

Reported event dataset (latitude and longitude coordinates)

Airport infrastructure datasets for the United States, Canada, and Australia

Methods & Tools

R for data cleaning and transformation

Custom Haversine distance calculations

Geospatial mapping with sf and ggplot2

Density estimation and layered visualization techniques

Comparative grouped bar chart analysis

Key Features

Multi-country dataset integration

Custom distance function implementation

Spatial density mapping

Infrastructure proximity analysis

Cross-regional comparative visualization
